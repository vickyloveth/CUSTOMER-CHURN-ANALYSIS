# Customer Churn Analysis (Excel Project)

## Project Overview
This project analyzes customer churn in a telecommunications company using Excel to uncover patterns, trends, and actionable insights.

The goal is to understand:
- How many customers churned
- How pricing affects customers
- Whether churn differs by gender
